# **App Name**: FileZen

## Core Features:

- File Upload: Allow users to securely upload files of various formats and sizes.
- Secure Sharing: Generate unique, secure links for each uploaded file, enabling controlled sharing.
- Link Expiration: Set expiration dates and times for shared file links.
- AI-Powered Suggestion Tool for Permissions: The sharing interface will feature a tool powered by a generative AI, which provides the user with suggestions about permissions and whether the recipient needs the link for View Only, Download, or Editor access. 

## Style Guidelines:

- Primary color: HSL(210, 65%, 50%) - A vibrant blue (#3391FF) conveys trust and reliability.
- Background color: HSL(210, 20%, 95%) - A very light blue (#F0F5FA) provides a clean and professional backdrop.
- Accent color: HSL(180, 55%, 55%) - A bright cyan (#4DD8D8) highlights interactive elements and calls to action.
- Body and headline font: 'Inter' - A versatile sans-serif font that works well for both headings and body text, ensuring readability and a modern aesthetic.
- Use flat, minimalist icons that clearly represent file types and sharing actions.
- Maintain a clean and intuitive layout with a focus on ease of navigation and clear visual hierarchy.
- Incorporate subtle animations for file uploads, link generation, and other interactive elements to provide user feedback.
